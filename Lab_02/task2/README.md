# Lab 2.2 — Custom JSX Elements & Dynamic Props

This folder contains the React project for **Lab 2.2** (Week 2): custom components, props, JSX expressions, fragments, and mapping collections.

## Project

- `react-props/` — Vite + TypeScript + React project

## What implements the tasks

- **Task 1 (Card: title + children + className)**: `src/components/Card.tsx` (used twice in `src/App.tsx`)
- **Task 2 (ProductList mapping + keys + expressions)**: `src/components/ProductList.tsx`
- **Task 3 (Section: fragment with title + children)**: `src/components/Section.tsx` (composed in `src/App.tsx`)

## How to run

1. Open a terminal in `Lab_02/task2/react-props`.
2. Install dependencies (once):

   ```bash
   npm install
   ```

3. Start the dev server:

   ```bash
   npm run dev
   ```

4. Open the printed local URL (for example `http://localhost:5173`).
