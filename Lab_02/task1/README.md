# Lab 2.1 — Fragment Drill & List Mapping

This folder contains the React project for **Lab 2.1** (Week 2): fragments, JSX expressions, and mapping arrays to lists.

## Project

- `react-fragments/` — Vite + TypeScript + React project

## What implements the tasks

- **Task 1 (fragment-only layout)**: `src/components/FragmentLayout.tsx`
- **Task 2 (map array to list + keys)**: `src/components/ItemList.tsx`
- **Task 3 (combined fragment + list + total)**: `src/components/ItemSummarySection.tsx`

## How to run

1. Open a terminal in `Lab_02/task1/react-fragments`.
2. Install dependencies (once):

   ```bash
   npm install
   ```

3. Start the dev server:

   ```bash
   npm run dev
   ```

4. Open the printed local URL (for example `http://localhost:5173`).
