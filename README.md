# React Fundamentals — Labs 01–02

This repository contains the solutions for **Lab 01** and **Lab 02**.

## Folder structure

- `task1/` (Lab 1.1)
  - `imperative.html` — jQuery (imperative) highlight toggle.
  - `react-highlight/` — Vite + TypeScript + React project implementing the declarative highlight toggle.
  - `README.md` — how to run both versions.
  - `comparison_report.md` — written comparison between imperative and declarative implementations.
- `task2/` (Lab 1.2)
  - `react-counter/` — Vite + TypeScript + React counter app.
  - `README.md` — setup and run instructions.
  - `view_layer_explanation.md` — explanation of React as the view layer.
- `Lab_02/task1/` (Lab 2.1)
  - `react-fragments/` — fragments + list mapping tasks.
  - `README.md` — run instructions + task/component mapping.
- `Lab_02/task2/` (Lab 2.2)
  - `react-props/` — Card, ProductList, Section tasks.
  - `README.md` — run instructions + task/component mapping.
- `AI_REPORT.md` — required report describing how AI tools were used.

## How to run the projects

### Lab 1.1 — Highlight toggle

- **Imperative (jQuery)**: open `task1/imperative.html` directly in a browser, then click **“Toggle Highlight”**.
- **Declarative (React)**:
  1. Open a terminal in `task1/react-highlight`.
  2. Run `npm install` (once).
  3. Run `npm run dev` and open the printed URL in the browser.

### Lab 1.2 — Counter app

1. Open a terminal in `task2/react-counter`.
2. Run `npm install` (once).
3. Run `npm run dev` and open the printed URL in the browser.

### Lab 2.1 — Fragments + list mapping

1. Open a terminal in `Lab_02/task1/react-fragments`.
2. Run `npm install` (once).
3. Run `npm run dev` and open the printed URL in the browser.

### Lab 2.2 — Custom components + props + mapping

1. Open a terminal in `Lab_02/task2/react-props`.
2. Run `npm install` (once).
3. Run `npm run dev` and open the printed URL in the browser.
